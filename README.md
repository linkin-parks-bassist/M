# M: The Everything Pedal

![Prototype Image](docs/resources/image.jpg)

**M** is a digital effects pedal. It is:

- Completely effortless to use. With its intuitive, smooth touch-screen UI, the workflow is simply: press buttons, drag and drop, turn dials, et cetera, and hear the magic. No technical knowledge required.

It is also:

- A fully programmable real-time digital signal procressing platform. With an A/B pair of fully pipelined, purpose built DSP cores running at 112.5MHz with single-cycle MAC throughput and an on-device, just-in-time compiler for its bespoke DSL, and dedicated DSP hardware including accelerated IIR engine and delay buffer controllers, any effect that can exist in theory can be created - with nothing but a text editor.

## Features

- Apply any effects, in any order. Effects can be added with a single touch and re-ordered with a simply tap-and-drag, and changes are applied in real-time. 
- Zero latency.... Approximately. Signals coming in are processed and sent back out in less than a tenth of a millisecond.
